HTML

HTML stands for HyperText Markup Language.It is the standard language used to create the structure of webpages.

HTML describes:-

->Text

->Images

->Links

->Forms

->Tables

->Multimedia content

Features of HTML:-

->Easy to Learn.

->Platform Independent.

->Support Multimedia.

->Used for webpage structure.

->Works with CSS and Javascript.

CSS

CSS stands for Cascading style sheets.It is used for styling and designing webpages.

CSS controls:-

->Colors

->Fonts

->Layouts

->Animations

->Responsive Design

Features of CSS:-

->Seperates design from structure.

->Reduces code repetition.

->Improves webpage appearance.

->Supports responsive layouts.
